Orchard FEM Ubuntu 24.04 dependency and validation guide
=========================================================

This file is the Ubuntu-focused dependency checklist for the active Orchard FEM workflow.

1. Required system packages
---------------------------
sudo apt update
sudo apt install -y git python3 python3-pip python3-venv

2. Python environment for the current repository tests
------------------------------------------------------
python3 -m venv .venv
source .venv/bin/activate
python -m pip install --upgrade pip
python -m pip install -e ".[ubuntu-test]"

What this installs:
- pytest
- pytest-cov
- ruff
- mypy
- numpy
- scipy
- pandas
- matplotlib
- pyyaml
- jupyterlab

3. Optional ML packages
-----------------------
python -m pip install -e ".[ml]"

Adds:
- torch
- h5py

4. Recommended FEniCSx / PETSc / SLEPc backend
---------------------------------------------
Recommended on Ubuntu: use conda or micromamba instead of pip.

Additional Ubuntu packages:
sudo apt install -y openmpi-bin libopenmpi-dev

Environment creation:
conda env create -f config/fenicsx_pinn_environment.yml
conda activate orchard-fenicsx

Key packages in that environment:
- dolfinx
- petsc4py
- slepc4py
- mpi4py
- numpy
- scipy
- pandas
- matplotlib
- pyyaml
- h5py
- pytest
- pytest-cov
- ruff
- mypy
- pytorch

5. Orchard FEM validation commands
----------------------------------
python -m orchard_fem doctor
python -m orchard_fem verify --skip-demo-suite
python -m orchard_fem full-validate --skip-fenicsx-tests --skip-python-demo-suite

Inside the PETSc/SLEPc environment:
conda run -n orchard-fenicsx python -m orchard_fem verify --skip-integration

6. Optional smoke runs
----------------------
python -m orchard_fem run examples/demo_orchard.json --output-csv build/demo_frequency_response.csv
python -m orchard_fem visualize examples/demo_orchard.json build/demo_frequency_response.csv --output-prefix build/demo_frequency_response
