# Archived C++ Headers

The active C++ headers have been removed from the main branch.
`include/` now remains only as an archive marker for the retired C++ implementation.

Its former conceptual surfaces have been migrated into Orchard FEM package namespaces such as:

- `orchard_fem.domain`
- `orchard_fem.branches`
- `orchard_fem.geometry_topology`
- `orchard_fem.discretization`
- `orchard_fem.solver_core`

See [cpp_migration_inventory.md](../docs/cpp_migration_inventory.md) for the mapping ledger.
